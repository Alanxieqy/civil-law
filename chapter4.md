# 债法 #

## 债法总论 ##

意定之债：

1. 合同
2. 单方行为（馈赠、捐助、单方允诺）
3. 共同行为（设立公司协议、合伙协议）

法定之债：

1. 授权行为
2. 无因管理
3. 缔约过失
4. 拾得遗失物
5. 亲属间的抚养请求权
6. 受益人的法定补偿责任

## 债的转移 ##

* 法定转移：

1. 代位求偿权
2. 继承
3. 买卖不破租赁
4. 房屋租赁合同的法定承受
5. 企业合并、分立

> 代位求偿权 ： 保险代位求偿权，是指当保险标的遭受保险事故造成的损失，依法应由第三者承担赔偿责任时，保险公司自支付保险赔偿金之日起，在赔偿金额的限度内，相应地取得向第三者请求赔偿的权利。“保险代位权是各国保险法基于保险利益原则，为防止被保险人获得双重利益而公认的一种债权移转制度”，通常认为保险代位权其实质是民法清偿代位制度在保险法领域的具体运用。

* 裁判转移
* 约定转移：

1. 债权让与
2. 债务承担
3. 约定概括承受

### 债权装让 ###

![债权转让](https://docs.google.com/drawings/d/12fiNw4cM4jVKaNsQ_6wqA0GtZtOOqYyC8RKgDLKpxLA/pub?w=480&h=360 "债权转让")

债权转让的要件如上图，甲为债务人，乙为债权人，乙将对甲的债权装让与丙

债权转让的构成要件有：

1. 债权具有可转让性。有三种债权不具有转让性：
   * 约定不得转让的。但是不得对抗善意受让人
   * 法律禁止转让的。如赡养费请求权；抚养费请求权
   * 依照性质不得转让的。如债权转让将导致债务人的给付内容完全变更
2. 让与人与受让人大城债权装让的合意。虽然乙将债权转让给丙时，应当通知甲，但是通知并非丙取得债权的必要条件。但是，不通知的，转让对甲不发生效力。只有通知到甲，债权转让才对甲有效力
3. 遵循法定形式。法律规定转让债权应当批准、登记手续的，依照规定

与债权转让相伴随的法律效果：

1. 从权利随同转移
   * 基于担保物权与保证债权的从属性，债权全部转让的，抵押权、质权、留置权、保证债券随同转移
   * 基于担保物权的不可分性，债权部分转移的，抵押权、质权仍同时担保已经转让部分的债权与尚未转让部分的债权
2. 债务人接到债权转让通知时，债务人对让与人的抗辩，可以向受让人主张
3. 被转让债权发生诉讼时效中断。应当认定诉讼时效从债权转让的通知到达债务人之日起中断
4. 债务人对债权受让人的抵消权，不要求“互负债务”这一要件，债务人对债权让与人之抵消适格地位不因债权让与而受影响

### 债务承担 ###

![债务承担](https://docs.google.com/drawings/d/1DzoqjD0RanwsRNSMkpnRF7d9z8scyf9WMByxxNVeLfs/pub?w=480&h=360 "债务承担")

**免责的债务承担**要件如上图，甲为债务人，乙为债权人，甲对乙债务全部或部分免责地转让给丙承担，有两种模式：

1. 第三人丙与债权人乙达成协议：“甲对乙的债务全部或者部分由丙承担”，这种模式无须甲同意，但应当通知甲。乙丙的协议属于“利益第三人合同”，甲可以放弃因此获得的利益
2. 甲与第三人达成协议，由丙来达成债务承担，其构成要件有四个：
    * 甲对乙的债务具有可转让性
    * 甲和丙就全部或者部分转让债务达成合意
    * 须经债权人乙同意。不经过债权人乙同意的，甲丙之间的协议有效，只是不对乙发生效力
    * 遵循法定形式

免责的债务承担及其伴随的法律效果

1. 就已经全部或者部分转让与受让人的债务，原债务人免除债务，原债务人也不能对受让人履行债承担担保责任
2. 抗辩的援用。债务受让人可以主张对债权人的抗辩
3. 被承担之债发生诉讼时效中断。从债务承担的意思到达债权人之日起中断
4. 第三人提供担保，未经其书面同意，债权人允许债务人转移全部或者部分债务的，担保人不再提供担保
5. 保证期间，债权人许可债务人转让债务的，应当取得保证人书面同意，保证人对未经其同意转让的债务，不再承担担保责任

![并存的债务承担](https://docs.google.com/drawings/d/1ZpDhB4F9abwpI4MQlffqW8jrmLpt4M0UPqOXDa1IyzA/pub?w=480&h=360 "并存的债务承担")

并存债务承担是指债务人并不脱离合同关系，而由第三人加入到合同关系当中，与债务人共同承担合同义务的债务承担方式。

1. 第三人承担债务原则上不需取得债权人的同意，债务人或者第三人向债权人发出债务承担的通知即可。
2. 第三人加入债的关系后，得以原债务人对抗债权人的事由对抗债权人，但不得以自己与债务人之间的事由作为抗辩理由对抗债权人。
3. 并存的债务承担成立后，债务因原债务人或者第三人的全部清偿而消灭，债务的消灭因第三人的清偿发生时，在第三人与债务人之间可能发生求偿关系。

### 合同权利义务的约定承受 ###

合同权利义务的约定承受是指合同当事人一方将自己的合同权利与义务一并一转给第三人

构成要件有：

1. 须有合法有效的双方合同存在
2. 合同当事人一方与第三方达成合同承受的协议
3. 经合同对方当事人同意

约定承受的效力：

1. 第三人取得让与人所享受的合同权利及其担负的合同义务
2. 让与人推出合同关系
3. 从权利、从债务随同移转，但专属于让与人的除外

**两个例外**

在旅游行程开始前的合理期限内，旅游者将其在旅游合同中的权利义务三人的，无需经合同双方当事人的同意

旅游服务经营者将其合同权利义务转让给第三人，须经旅游者的同意。在旅游过程中，旅游者遭受损害，请求与其签订旅游合同的旅游经营者和实际提供旅游服务的旅游经营者承担连带责任的，人民法院予以支持

## 不当得利 ##

### 不当得利的构成要件 ###

1. 一方获得利益（包括财产积极增加和财产消极增加）。积极增加：本不应该增加的增加了；消极增加：本应该减少的没有减少
2. 他方受有损失（包括财产积极减少和财产消极减少）。积极减少：本不应该减少的减少了；消极减少：本应该增加的没有增加
3. 获得利益与收到损失之间具有因果关系。在给付型不当得利中，给付关系替代因果关系
4. 获得利益没有法律上的原因

### 不当得利的排除情形 ###

给付型不当得利的排除情形

法律规定受害人不得请求返还：

1. 给付系履行道德义务
2. 债务人为清偿未到期债务而给付的
3. 因清偿债务而给付，于给付时知道无给付义务的
4. 因不法原因而给付的，但不法原因只存在受领一方的除外（对绑架者支付赎金、索贿）

其他排除情形：

1. 强迫得利。受损人因其行为使受益人受有利益，但违反了受益人的意思，不符合其经济计划的情形
2. 反射得利。一方因一定的行为或实施而受益，但并未导致他人利益损害的情形，其不构成不当得利
3. 非财产性受益。不当得利属于债法范畴，仅调整财产利益关系

### 不当得利之债 ###

成立不当得利之债的，受害人有权请求受益人返还不当得利，返还范围是所受利益和孳息：

1. 原物存在的返还原物；原物不存在的，折价返还；原物损毁后存在代位物的，应当返还代位物
2. 所有利益依其性质不能返还的，应返还其价额
3. 所有利益产生的孳息（自然孳息和法定孳息），应当返还
4. 受理人额利用不当得利所取得的其他利益（投资所产生的利益），扣除劳务管理费用后，应当予以“收缴”，而不是返还给受害人

受益人善意与恶意对返还范围的影响

对此,我国民法暂无规定。根据民法理论,其规则如下：

1. 受益人为善意(指得利时,不知道也不应当知道受有不当得利)的,仅返还现存利益,对已经不存在的利益不负返还义务。 
2. 受益人为恶意(指得利时,知道或者应当知道受有不当得利)的,须返还所受利益,对已经不存在的利益亦负返还义务。
3. 受益人在取得利益时为善意,嗣后变更为恶意的,其返还范围应以恶意开始之时存在的利益为准。

不当得利由第三人无偿取得时的返还义务 

* 善悫的受益人将所受利益无偿让与第三人：

1. 善意受益人就无偿让与第三人的部分不负返还义务(因系无偿让与,现存利益不存在。须注意：若是有偿让与,则现存利益仍然存在,受益人仍负返还不当得利的义务。 
2. 第三人在受益人免除返还责任的限度内对受损人负返还的责任。

* 恶忘的受益人将所受利益无偿让与第三人：

1. 恶意受益人的返还义务不因此而受影响。原因：恶意不当得利人应返还所受利益, 即使所受利益已不存在,恶意得利人的不当得利返还义务不受影响。
2. 受害人享有选择权,亦可要求第三人负担返还义务。当然,这需要受害人与第三人之前符合不当得利的构成要件。

## 保证 ##

![保证](https://docs.google.com/drawings/d/1l9ZN5jgq5rQl4aCXQMfg9-5TAyOApWAgkUwNOv_Uzd4/pub?w=480&h=360 "保证的概念")

保证合同：保证人与债权人约定在债务人不履行债务时，由保证人按照约定履行债务或者承担责任的协议

1. 保证合同的当事人是债权人和保证人
2. 保证人须为债务人以外的第三人
3. 债务人不是保证合同的当事人

保证合同成立的方式：

1. 订立书面保证合同
2. 主合同定有保证条款，保证人在主合同上签字或者盖章的
3. 主合同虽无保证条款，保证人在主合同上以保证人身份签字或者盖章的
4. 第三人单方以书面形式向债权人出具保证书，债权人接受并且未提出异议的
5. 自然人之间的口头保证合同，有两个以上无利害关系人证明的，也可视为保证合同成立

保证债务的内容：

1. 代为履行。在债务人不对债权人履行非金钱债务时，由保证人按照约定代为履行
2. 赔偿责任。不能代为履行，或者按照约定不代为履行的，由保证人承担赔偿责任。（补充责任）

保证责任的范围：

1. 有限保证。按照**约定**，保证人仅对债务人的部分债务承担保证责任
2. 无限保证。保证人对债务人的全部债务承担保证责任，未约定保证范围或约定不明的，保证人须对主债权及其利息、违约金、损害赔偿金和实现债权的费用承担保证责任

保证债权的从属性

1. 成立上的从属性
2. 效力上的从属性
3. 内容和范围上的从属性
4. 消灭上的从属性
5. 转以上的从属性

保证期间，债权转让的，保证债权同时转让，除非规定禁止债权转让或者仅对特定债权人承担保证责任的

保证期间，债务人经债权人同意转让债务的，应当取得保证人书面同意，保证人对未经其同意转让部分债务，不再承担保证责任

保证期间，债权人对债务人协议变更主债务的，应当虚的保证人书面同意，未经同意的：

1. 减轻债务的，保证人对及减轻后的债务承担保证
2. 增加债权的，保证人对加重的部分不承担保证
3. 变动债务履行期间的，保证期间仍为原合同约定或者法律规定的期间

### 保证方式 ###

#### 一般保证 ####

保证人仅对债务人不能履行的债务承担补充责任的保证。

    在司法考试中，题目中的“不”即是“不能”，一般都认为是一般保证，特殊标明的例外
    
一般保证必须明确约定。当事人没有约定或者约定不明确的，按照连带责任保证承担保证责任

**一般保证人享有先诉抗辩权，在主合同纠纷未经审判或者仲裁，并就债务人财产依法强制执行仍不能履行债务前，对债权人可以拒绝承担担保责任**

丧失先诉抗辩权的情形：

1. 债务人住所变更，知识债权人要求其履行债务发生重大困难
2. 债务人下落不明，移居境外，且无财产可供执行
3. 人民法院受理债务人破产案件，中止执行程序
4. 保证人以书面形式同债权人或者债务人放弃先诉抗辩权的

#### 连带责任保证 ####

保证人在债务人不履行债务时与债务人承担连带责任的保证，连带责任保证人不享有先诉抗辩权

#### 保证人的诉讼地位 ####

一般责任：

1. 可以只列债务人为被告
2. 不能只列保证人人被告，只列保证人为被告的，法院应当追加债务人为共同被告
3. 将债务人和保证人列为共同被告
4. 在判决中明确在对债务人财产依法强制执行后仍不能履行债务时，有保证人承担保证责任

连带责任：

1. 可以只列债务人为被告
2. 可以只列保证人为被告
3. 可以将债务人和保证人列为共同被告

### 共同保证 ###

同一笔债，只有一个保证人担保的，为单独保证；由两个以上保证人共同保证的，为共同保证

**按份共同保证**

保证人与债权人约定了各保证人承担保证责任的份额

* 保证人按照确定的份额对债权人承担担保责任
* 保证人之间无须内部关系
* 保证人承担保证责任后，只能向债务人追偿，无权请求其他保证人分担

**连带共同承担**

成立方式：

1. 明确约定各保证人彼此承担连带责任
2. 均未与债权人约定各保证人承担保证责任的份额

效力：

* 债务人不履行到期债务时，各保证人应对债权人承担连带责任
* 保证人的追偿有顺序性：先向债务人追偿；不能追偿的部分，由各连带保证人按其内部约定的比例分担，没有约定的，平均分担

### 保证期间 ###

一般保证的保证人与债权人未约定保证期间的，保证期间为主债务履行期届满之日起6个月

在合同约定的保证期间和法律规定的保证期间，债权人未对债务人提起诉讼或者申请仲裁的，保证人免除保证责任

保证人对债权人的保证债务受三重时间的限制：

1. 保证期间
2. 保证债务诉讼时效期间
3. 主债务诉讼时效期间

保证人有脱逃保证责任的机会：

1. 第一次 :债权人未在保证期问内依法定方式主张权利 (一般保证 :债权人未起诉或申请仲裁 ;连带责任保证 :债权人未请求保证人承担保证责任)。 保证责任因保证期间 经过而消灭。
2. 第二次 :保证人对债权人之保证债务的诉讼时效期间经过，保证人获得自己的抗辩权。
3. 第三次 :主债务的诉讼时效期间经过。此时,保证人有权援用债务人的抗辩权 (这一援用来源于保证债务在内容和范围上的从属性)。

保证期间的作用：督促债权人依法定方式积极行使债权

在保证期间内，若债权人未依法定方式主张权利，保证期间经过，保证责任消灭

保证期间的确定

* 有约定的按照约定
    - 约定的保证期间早于或者等于主债务履行期限，视为没有约定，按照6个月计算
    - 约定保证人承担保证责任直至主债务本息还清为止，视为约定不明，保证期间为主债务履行期届满起2年
* 未约定保证期间的，不论一般保证还是连带保证，保证期间均未6个月
    - 自主债务履行期届满之日起算
    - 没有约定债务履行期限的，自宽限期限届满之日起算
    - 债务人被宣告破产的，自破产程序终结之日起算
